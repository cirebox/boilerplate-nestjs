# API boilerplate-nestjs

Projeto base para api usando nestjs

## Installation
```bash
$ pnpm install
```

## DB Generate
```bash
$ npx prisma db push
```

## Seed
```bash
$ npx prisma db seed
```

## Running

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# unit watch
$ pnpm run test:watch

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
